# 排序算法面向问题

当整数数组A排序时，涉及A的问题：

- 在数组A中搜索特定值v

- 查找静态数组A中的最小、最大、第k个最小、最大值

- 测试唯一性并删除数组A的重复项

- 计算特定值v在数组A中出现多少次

- 设置数组A和另一个排序数组B之间的交集、联合

- 寻找一个目标对x属于A和y属于A，使得x+y等于目标z
